## Running locally

### On GitHub:

:::info
With GitHub, you can edit the wiki directly in your browser. This is the easiest way to contribute.
:::

:::warning
you will not see a preview with fumen embeds or minos
:::

1. Fork this repository (click the edit this page button)
2. Make your changes
3. Submit a pull request

### With preview server:

1. Install [Node.js](https://nodejs.org/en/download/)
2. Clone this repository
3. Run `npm install` in the repository directory
4. Run `npm run start` to start the development server
5. Open http://localhost:3000 in your browser
